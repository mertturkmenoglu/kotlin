**Problem**: Given two strings *s1* and *s2* , write a function to determine if *s2* is an anagram of *s1*.

**Example 1**:
```
Input: s1 = "emily", s2 = "iemyl"
Output: true
```

**Example 2**:
```
Input: s1 = "abc", s2 = "def"
Output: false
```

**Approach**: Hash Table
* String contains only lowercase English alphabet characters
